BORDERS

We can add borders to our webpage using the stylesheets.
We can do this by using the (border) element.
We can change the size(border-width/border-height) style(border-style) and color(border-color)of the borders.

MARGINS:

Margin is the space of the text outside the border.
It decides how much the text will be away from each side of the page.

EXAMPLES:

This text has a margin of 100px only from the top.

This text has a margin of 50px from all four sides.

PADDING:

Padding is the distance between the border and the text.

EXAMPLE:

This text has a padding of 50px only from the top and the bottom.

This text has a padding of 80px.
