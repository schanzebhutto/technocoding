INTRODUCTION


HTML

  • Html stands for HyperText Markup Language.
  • It was found by Tim Berners-Lee in 1991.
  • It is used for front-end website development.
  • It is responsible for the content of the website.

CSS
  • CSS stands for Cascading Style Sheet.
  • It was found in 1996.
  • It is used for front-end website development.
  • It is responsible for the visuals and layout of the website.
