LET'S BEGIN!


To change the text formatting of the text, we use the internal CSS option.

NOTE:All the features mentioned below, will be formatted under the (style) tag


FONT SIZE:

We can change the font size, either in the form of pixels or in the form of percentage.

EXAMPLE:

PIXELS:This is the size of 5px

PERCENTAGE:This is the size of 10%


FONT FAMILY:

There are different types of font families which we can use in our HTML code.

EXAMPLES:

CURSIVE: This is the "cursive" font.

SANS-SERIF: This is the "sans-serif" font.

MONOSPACE: This is the "monospace" font.


FONT COLOR:

We can also change the font color of the text, using the (color) element.

EXAMPLES:

The font color is green.

The font color is blue.

The font color is Red.


TEXT ALIGNMENT:

We can also align the text, using the element (text-align).

EXAMPLES:

Left align

Center align

Right align


TEXT DECORATION:

By using the (text-decoration) element, we can format the text.

EXAMPLES:

underline

linethrough

overline

We can transform the text using the (text-tranform) element.
