NOTE: We can add the absolute and relative paths by using the (a href) element.

ABSOLUTE PATH:

Absolute paths are links that contain the complete address to get to a webpage.

EXAMPLE:

DAWN NEWS: dawnnews


RELATIVE PATH:

Relative paths are the links that are provided when the files of the webpage are in
the same folder as the page displaying the link.

EXAMPLE:

kindly visit our pevious lecture: session1
